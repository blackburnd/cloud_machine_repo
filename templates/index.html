Embracing the AI Revolution

Things are changing fast, and it's a terribly exciting time to be skilled in software development—especially when you know what "correct" looks like. I liken using AI as a pair programmer to having my own "honesty-impaired Djinn" for software. It's constantly lying, but AI can absolutely provide valuable solutions with very explicit instructions on guardrails, oversight, and follow-up.

With proper constraints, I've learned to create some pretty cool solutions—but only if AI has a short leash. Letting it run wild is a mistake and a new type of frustrating experience that expands both vocabulary and patience.

This Site: A Technical Showcase

As an academic exercise born from the need to advocate for my skillset beyond what a simple resume could offer, I created this OAuth2-authenticating, LinkedIn-integrated site. This project allowed me to immerse myself in Google's Cloud Platform, VSCode, CI/CD, and Copilot agents to advertise my skills in depth.

This site is the result of a few weeks of that immersion—a perfect opportunity to learn new technologies and framework integrations. The site runs on Google Cloud Platform with a virtual instance, backed by a PostgreSQL cloud instance, and topped off with CI/CD integration configured to a public GitHub repository.
