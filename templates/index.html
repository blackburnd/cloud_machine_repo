This portfolio site is the result of that imersion to advertise some of my abilities more in depth than just a resume submission, so this was a perfect opportunity to learn a handful of new technologies, and framwork integrations. In fact this site, is the imersion project. The domain was purchased from squarespace, routed to Google Cloud flask based postgresql backed python app that uses google oauth2 for authentication and authorization to admin mode. Nginx handles the middleware, and intetgration with google's gmail API allows the contact form emails to be sent. The site is runnin GCP on a virtual instance, with CI/CD integration configured to the public repo on github.
